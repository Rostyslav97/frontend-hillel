Діалогові функції в JavaScript

JavaScript надає кілька функцій для взаємодії з користувачем через діалогові вікна.

  • alert(): Показує діалогове вікно з повідомленням та кнопкою OK.
  • prompt(): Відображає діалогове вікно з текстовим полем для вводу користувачем та повертає введене значення.
  • confirm(): Показує діалогове вікно з повідомленням та двома кнопками: OK та Скасувати, повертає true, якщо натиснуто OK.

Консоль у браузері як інструмент для розробників

Консоль у браузері є потужним інструментом для розробників, що дозволяє виконувати код JavaScript в реальному часі, а також слугує інструментом для логування, дебагу та інтерактивного тестування веб-додатків.

Основні функції консолі:

  • console.log() - Виводить повідомлення в консоль для логування інформації.
  • console.error() - Виводить помилкове повідомлення в консоль.
  • console.warn() - Виводить попереджувальне повідомлення в консоль.
  • console.info() - Виводить інформаційне повідомлення в консоль.
  • console.debug() - Виводить повідомлення для дебагінгу в консоль.
  • console.clear() - Очищує консоль від попередніх повідомлень.
  • console.group() і console.groupEnd() - Дозволяють групувати повідомлення в консолі.
  • console.table() - Виводить дані у вигляді таблиці в консолі.
  • console.assert() - Виконує асерцію і виводить повідомлення, якщо умова хибна.
  • console.time() і console.timeEnd() - Дозволяють виміряти час виконання коду.
