Підключення JavaScript до сторінки

Для підключення JavaScript до вашої сторінки використовуйте тег <script>. Є декілька способів це зробити:

Вкладений JavaScript

Ви можете вбудувати JavaScript безпосередньо в HTML-код за допомогою тега <script> у розділі <head> або <body>.

Зовнішній JavaScript файл

Також ви можете створити окремий JavaScript файл з розширенням .js та підключити його до вашого HTML за допомогою тега <script src="шлях_до_файлу.js"></script>.
